For artists who want to explore code as a medium.

For designers who want to create interactive user experiences.

For programmers who want to build creative applications.

If you want to learn how to express your creative ideas by writing computer programs, then this workshop is for you.

You will learn to

  • develop your practical programming skills
  • use algorithms to make animated gifs
  • build interactive websites
  • explore glitch art
  • perform face tracking on a live webcam
  • turn data into sound
  • create your own music visualizer

To do this, we will be using the programming language Processing and its web interpertation p5.js.

How many. Not more than 10 particpants.
How much. 500 LE.

FAQ

Do I need to be a programmer to join this workshop?
No. But you need to have no fear of writing code.

Do I need to be an artist to join this workshop?
No. But you need to have no fear of making art.

Computer requirements?
Your own laptop running Windows, Mac OS X, or Linux.

What can my final project be?
Final projects will be personal creations. They can be whatever you want to use creative coding for.
A gif. Graphic prints. An animation. A video. A song. A webpage.
